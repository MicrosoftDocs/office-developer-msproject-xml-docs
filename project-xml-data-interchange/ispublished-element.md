# IsPublished Element




Indicates whether the task is published.

    <IsPublished>
      BooleanValue
    </IsPublished>

## Example

The following example uses the IsPublished element to indicate that the task has been published.

``` xml
<Task>
  …
  <IsPublished>1</IsPublished>
  …
</Task>
```
